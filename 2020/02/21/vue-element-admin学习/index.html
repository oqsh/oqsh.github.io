使用vue-element-admin中的组件

  1. vue-element-admin的组件大部分都位于src/components/目录下，直接过去复制粘贴到自己的项目对应路径下即可，然后使用的时候记得import进来
  2. 还有一些组件是需要自己使用npm安装的，比较好的方法是直接去/package.json文件里面写好的自己缺的组件，然后执行npm install即可
